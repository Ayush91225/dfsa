## Features Overview

### Class Creation
- Simple classroom creation with just a class name
- Designed for physical classroom management

### Student Management
- Add students by email address
- View and manage enrolled students
- Remove students from classes

### Subject Management
- Create subjects with descriptive names
- Upload teaching materials (PDFs and images)
- Visual file management with type-specific icons

### Material Organization
- Card-based layout for easy browsing
- Subject-specific color coding
- Quick access to lesson materials
